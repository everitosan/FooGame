# Foo Game
This is a prototipe of a game developed for FooStudio.

## Requirements
 - Google Chrome
 - npm
 - bower

## To start the game
Navigate to the root dir of the project by the terminal and type:
```sh
$ npm start
```
That should take care of the dependencies installation and code compilation.

## How to play
 - a | ← - Move to Left
 - d | → - Move to Right
 - s | ↓ - Move Down
 - w | ↑ - Move Up
 - space - Jump
 - ↵   - Shoot
